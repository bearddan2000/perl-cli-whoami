# perl-cli-whoami

## Description
This is a POC project to demonstrate
a perl console app.

## Tech stack
- perl

## Docker stack
- docker-cli
- alpine:latest
- centos:latest
- ubuntu:latest

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
```sudo ./install.sh -u```

## To stop (optional)
```sudo ./install.sh -d```
